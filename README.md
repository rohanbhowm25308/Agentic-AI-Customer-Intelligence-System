#  Project Overview

The **Agentic AI Customer Intelligence System** is an end-to-end AI-powered analytics platform designed to help businesses analyze customer datasets effortlessly.

Users simply upload a CSV dataset, and the platform automatically performs data preprocessing, customer segmentation, business analytics, AI reasoning, visualization, intelligent chat, and PDF report generation.

#  Installation

Clone the repository

```bash
git clone https://github.com/rohanbhowm25308/Agentic-AI-Customer-Intelligence-System.git
```

Move into the project

```bash
cd Agentic-AI-Customer-Intelligence-System
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the FastAPI server

```bash
uvicorn main:app --reload
```

Open your browser

```
http://127.0.0.1:8000
```
